# Intelligent Log Anomaly Detection System

This project focuses on building an AI-based system for monitoring web application logs and detecting abnormal system behavior.

The system collects information such as API response time, request volume, error rate, login activity, and server status. Machine learning models are then used to identify unusual events, including:

* Sudden increases in traffic
* High API response time
* Repeated login failures
* Server error spikes
* Database timeout errors
* Suspicious user activity

The main goal of the project is to help developers detect application problems early and understand the possible causes of each incident.

## Main Technologies

* Python
* FastAPI
* Pandas
* Scikit-learn
* Isolation Forest
* Streamlit
* Docker
